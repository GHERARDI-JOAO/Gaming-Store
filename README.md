# Gaming Store

Bem-vindo ao **Gaming Store**, um projeto desenvolvido como parte do Processo de Trainees da Emakers Jr. Este projeto tem como objetivo criar um site funcional e responsivo para o universo gamer, utilizando tecnologias modernas de desenvolvimento Front-End.

## Tecnologias Previstas
- **HTML5** e **CSS3**: Para estruturação e estilização das páginas.
- **JavaScript (ES6+)**: Para adicionar interatividade ao site.
- **React**: Biblioteca JavaScript para construção de interfaces dinâmicas e reutilizáveis.
- **Git e GitHub**: Para controle de versão e hospedagem do repositório.

## Objetivos do Projeto

Este projeto vai ser desenvolvido para aplicar conceitos fundamentais do Front-End, incluindo:

- Criação de telas responsivas e adaptáveis a diferentes dispositivos.
- Implementação de formulários e navegação dinâmica.
- Utilização de práticas modernas de desenvolvimento com React.
- Organização de um repositório no GitHub para compartilhamento e revisão de código.

## Funcionalidades Planejadas

- Página inicial com um carrossel de destaques.
- Formulários de Login, Registro e Recuperação de Conta.
- Barra de navegação funcional e adaptativa (menu sanduíche em dispositivos móveis).
- Rodapé com links úteis.
- Utilização de `useState` para adicionar interatividade (ex.: alternador de tema ou contador).

## Responsividade e Design

O design do site será responsivo, garantindo uma boa experiência em dispositivos móveis, tablets e desktops. Para isso, utilizaremos práticas modernas de CSS e bibliotecas auxiliares, como o **Material UI**.

## Próximos Passos

1. Estruturar as páginas básicas com HTML.
2. Adicionar estilos iniciais com CSS.
3. Integrar funcionalidades interativas usando JavaScript e React.
4. Garantir a responsividade para diferentes tamanhos de tela.
